# Product Manager Job Application Kit (VMock)

This folder contains ready-to-edit documents and website code to help you apply for the Product Manager role shown in your screenshot.

## Files Included

1. `cv_saharsh_srivastava.md` - ATS-friendly one-page CV template tailored for Product Manager roles.
2. `cover_letter_vmock.md` - Targeted cover letter for VMock Product Manager position.
3. `portfolio_website/index.html` - Single-page portfolio website code.
4. `portfolio_website/styles.css` - Styling for the portfolio page.
5. `github_profile_README.md` - Suggested GitHub profile README content.
6. `application_checklist.md` - Submission checklist for CV, cover letter, website, and links.

## How to Use

- Replace all placeholders like `[X years]`, `[metric]`, and `[Project Name]` with your actual details.
- Export CV and cover letter to PDF before uploading.
- Host website using GitHub Pages or Netlify.
- Add website and GitHub links in the application form.

## Quick Publish (GitHub Pages)

1. Create new repo: `saharsh-portfolio`
2. Upload contents of `portfolio_website/`
3. Go to Settings -> Pages
4. Source: Deploy from branch (`main` / root)
5. Website URL will be available in 1-2 minutes.
